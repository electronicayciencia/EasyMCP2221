=====================================================
Easy MCP2221
=====================================================


Python interface for MCP2221(A).

MCP2221 is a simple 14 pin USB chip with

- 4 GPIO
- 3 channel ADC
- 1 DAC
- I2C
- UART
- Clock Output
- USB Wake-up via Interrupt Detection.
